# Canvas Pong

## Online demo
https://pong.tkarimdesign.com/

## Play locally via Docker or Node

### Docker

```shell
$ docker build -t canvas-pong .
$ docker-compose up
```

### Node

```shell
$ cd /path/to/canvas-pong
$ npm install
$ npm run dev
```

Local game can be found on http://localhost:8080
